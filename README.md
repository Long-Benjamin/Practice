# Practice
【主要功能】

1、点击图片查看时，图片带点击放大效果，并无缝进入第二个页面查看大图，支持手势缩放，切换上下图片

2、使用Zxing 的二维码生成和识别库，需要在列表点击事件中放开跳转代码
